# Slumber Secrets

This is the official landing page for the "Slumber Secrets" digital product, a guide to 7 nightly rituals for a deeper, more restorative sleep.

## About the Project

This project is a simple, effective landing page built with Next.js and Tailwind CSS. It is designed to present the product, showcase testimonials, answer frequently asked questions, and guide users through a manual purchase process via WhatsApp.
